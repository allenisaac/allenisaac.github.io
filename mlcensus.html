    As this is an issue of classification, the traditional machine learning models that will be used are SVM, random forest, K nearest neighbors. Through various techniques, these models will use our given variables to try to correctly classify the income of an individual. describe the three of them and the ideas of the parameters

    A big issue that was prevelant in all of the models, was an issue of overfitting. The issue with overfitting comes when there are too many features. These features can add insignificant or repetetive information. By using an ANOVA to F-test each variable, we are able to find whether features are significant, within a p-value of 0.05, and remove accordingly. As the qualitative features can't be left as they are we must first add one-hot features, essentially accounting for each possibiloity of the feature.
